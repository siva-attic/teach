Install guide

Step-by-step for your operating system

Select your OS below. Follow each step in order — the order matters because Maven uses the JDK.

1
Install Homebrew — the Mac package manager. If you already have it, skip to step 2.
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2
Install JDK 21 via Homebrew. This downloads and installs OpenJDK 21 automatically.
$ brew install openjdk@21
3
Link the JDK so macOS can find it. Homebrew prints this command after the install — copy it from there.
$ sudo ln -sfn /opt/homebrew/opt/openjdk@21/libexec/openjdk.jdk \ /Library/Java/JavaVirtualMachines/openjdk-21.jdk # Intel Mac path: /usr/local/opt/openjdk@21/...
4
Install Maven — Homebrew handles this too.
$ brew install maven
5
Install IntelliJ IDEA — search for "IntelliJ IDEA Community" on the JetBrains website and download the macOS version. The Community edition is completely free.
1
Install JDK 21 using winget (built into Windows 11). Run PowerShell as Administrator.
> winget install Microsoft.OpenJDK.21
2
Set JAVA_HOME — Maven needs this environment variable to find your JDK. Open System Properties → Environment Variables and add:
// Variable name: JAVA_HOME // Variable value (adjust path if different): C:\Program Files\Microsoft\jdk-21.0.x.y-hotspot
3
Install Maven — download the binary zip from the Apache Maven website, extract it (e.g. to C:\maven), then add its bin folder to your PATH.
// Add to PATH in Environment Variables: C:\maven\apache-maven-3.9.x\bin
4
Install IntelliJ IDEA — search for "IntelliJ IDEA Community" on the JetBrains website and download the Windows installer (.exe). Run it and follow the defaults.
5
Restart your terminal after setting environment variables — the changes don't apply to already-open windows.
1
Install JDK 21 — command differs by distro. Ubuntu/Debian:
$ sudo apt update $ sudo apt install openjdk-21-jdk # Fedora/RHEL: $ sudo dnf install java-21-openjdk-devel
2
Verify JAVA_HOME is set. If not, add it to your ~/.bashrc or ~/.zshrc.
# Add to ~/.bashrc export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64 export PATH=$JAVA_HOME/bin:$PATH $ source ~/.bashrc
3
Install Maven via your package manager.
$ sudo apt install maven # Fedora/RHEL: $ sudo dnf install maven
4
Install IntelliJ IDEA — the easiest way on Linux is via the JetBrains Toolbox App. Download it from the JetBrains website and it manages updates automatically. Alternatively, install from your distro's Snap store: sudo snap install intellij-idea-community --classic
Verification

Prove it works — three commands

Run these in your terminal. Each should print a version number. Click a command block to mark it done as you go.

terminal
CHECK 1 — Java runtime click to mark done ✓
java --version
openjdk 21.0.x 2024-01-16 LTS
OpenJDK Runtime Environment (build 21.0.x+...)
✓ You want to see "21" — anything lower and Spring Boot 3.x won't start.
CHECK 2 — Java compiler click to mark done ✓
javac --version
javac 21.0.x
✓ Must match your runtime. If this fails, you installed JRE instead of JDK — you need the full JDK.
CHECK 3 — Maven build tool click to mark done ✓
mvn --version
Apache Maven 3.9.x (...)
Maven home: /usr/local/Cellar/maven/3.9.x/libexec
Java version: 21, vendor: ...
✓ Critical: the "Java version" line here must also show 21. If it shows a different version, Maven is pointing to the wrong JDK.
🔧
mvn shows the wrong Java version? Maven reads the JAVA_HOME environment variable. Set it to your JDK 21 path and restart your terminal. Run echo $JAVA_HOME to see what it's currently set to.
